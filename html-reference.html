A valid HTML page will have <!DOCTYPE html>, head, html, and body.

HTML

Syntax

<html>,</html>

Description

The HTML tag tells the browser that this is an HTML document.

Example and Display

<html>

blahblahblah

blahblah

</html>

Header

Syntax

<head>,</head>

Description

The head tag is a container for all head elements.

Example and Display

<head>

(head elements)

</head>

Title

Syntax

<title>,</title>

Description

The title tag defines the title in the browser toolbar.

Example and Display

<title>blah</title>

Body

Syntax

<body>,</body>

Description

The body tag contains all the contents of the document.

Example and Display

<body>

afjsaodfjaosifjdiasfjosdfoi

</body>

Heading

Syntax

<h1> through <h6>,</h1> through </h6>

Description

The heading tags are used to define HTML headings. h1 is biggest, h6 is smallest.

Example

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>...</h3>

Display

Heading 1

Heading 2

...

Paragraph

Display

Lists

Syntax

<ul>,<ol>,<li>,</ul>,</ol>,</li>

Description

Unordered list, ordered list, list item

Example

<ol>
<li>Cat</li>
<li>Dog</li>
</ol>
<ul>
<li>Cat</li>

</ul>

Tables

Syntax

<table>, <tr>, <th>, <td>, </table>, </tr>, </th>, </td>

Description

defines table, table row, header cell, standard cell

Example

<table>
<tr>
<th>Name</th>
<th>Age</th>
</tr>
<tr>
<td>John</td>
<td>20</td>
</tr>
<tr>
<td>Bob</td>
<td>16</td>
</tr>
</table>

Display

Name Age
John 20
Bob 16

<p>This is a paragraph.<p>
</div>

Display

This is a paragraph.
